Menu Driven Game Page:- Game Corner

  Menu-Driven Program is a program that gets input from a user by showing the options list, known as the menu,   from which the user chooses their option.This project includes a menu-driven website which has been developed   using Css, Html and Javascript which allows the user to choose from 4 games.

  The Four games included are:

    1. FLIP IT:   It contains a grid of tiles , all tiles face down. Select two cards. If the two flipped cards are same                         they remain unturned or else they flip-down after a short time.

    2. SNAKEMANIA:   The game contains two components- snake and fruit.In the game of Snake, the player uses                                      the arrow keys to move a "snake" around the board. As the snake finds food, it eats the food,                                      and thereby grows larger.The game ends when the snake either moves off the screen or moves                                      into itself.

    3. FALLING BALL GAME:   It contains endless platforms with holes and a ball.The ball rolls down the platform                                                   from in between the gaps.The game ends if the ball touches the top border.

    4. BRAIN GAME:   It contains multiple choice questions with four options.A time limit of 15 seconds is given for a                                     particular question.Within the alloted time,the player has to answer the questions.
